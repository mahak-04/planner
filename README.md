# Monthly Task Planner 

A personal, aesthetic monthly task and habit tracker built using **HTML**, **CSS**, and **JavaScript**.

This planner is designed for daily use to track habits, tasks, sleep, and overall monthly progress in a calm, visually pleasing way.

---

## Features

- Monthly task grid 
- Click-to-complete daily habits
- Editable task names
- Visual progress charts
- Sleep tracking
- Data saved automatically using browser localStorage
- Fully client-side (no login)

---

## Built With

- HTML
- CSS (Tailwind + custom styles)
- JavaScript
- Chart.js

---

## How to Use

1. Open the app in a browser  
2. Click circles to mark tasks as complete  
3. Click task names to rename them  
4. Log daily sleep hours  
5. Progress updates automatically  

Your data is saved in the browser, so it persists between visits.

---

## Deployment

This project is intended to be hosted using **GitHub Pages** so it can be accessed like a web app.

---

## Notes

- Best viewed on desktop or tablet
- Data is stored locally in the browser
- Clearing browser data will reset progress
